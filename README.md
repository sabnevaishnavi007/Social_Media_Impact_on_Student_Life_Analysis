# Social Media Impact on Student Life Analysis

## 📌 Overview
This project analyzes how social media usage affects students' lifestyle, sleep, and productivity.

## 🛠 Tools Used
- Excel / Python
- Pandas, NumPy, Matplotlib

## 📊 Key Analysis
- Screen time vs productivity
- Social media usage vs sleep patterns
- Behavioral trends of students

## 📁 Dataset
Contains student data including usage hours, sleep time, and productivity levels.

## 🚀 Work Done
- Data cleaning and preprocessing
- Data visualization
- Trend and pattern analysis

## 📈 Conclusion
Excessive social media usage negatively impacts productivity and sleep quality.
